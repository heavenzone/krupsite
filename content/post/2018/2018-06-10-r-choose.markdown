# 计算出组合数


```r

```

```{.code-output}
## [1] 10
```

# 计算出排列数


```r
hg_p <- function(n, r) {
  choose(n,r)*factorial(r)
}
hg_p(5,2)
```

```{.code-output}
## [1] 20
```



```r

```
